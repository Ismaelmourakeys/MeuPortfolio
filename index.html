Projetos

Landing Page – MyGluco

Landing page desenvolvida para divulgação e apresentação do sistema, do aplicativo MyGluco, com foco em usabilidade, clareza de informação e experiência do usuário (UX). A página tem como objetivo principal atrair e engajar usuários sobre para a concientização da diabetes, apresentando suas funcionalidades, benefícios e diferenciais de forma organizada e visualmente atrativa.

Ferramentas utilizadas

Tecnologias e ferramentas empregadas neste projeto.

HTML
Tailwind
JavaScript
Firebase

Breve descrição: Estrutura com HTML, estilos com Tailwind via CDN e comportamento com JavaScript; backend/serviços com Firebase.

Aplicativo Mobile – MyGluco

Aplicativo mobile desenvolvido com foco na gestão de diabetes, permitindo aos usuários monitorar seus niveis de glicose, registrar refeições, atividades físicas e medicamentos. O app oferece gráficos e relatórios para auxiliar no controle da condição, além de notificações personalizadas para lembrar os usuários de suas rotinas de saúde.

Ferramentas utilizadas

Tecnologias e ferramentas empregadas neste projeto.

React Native
Node.js
NPM
Firebase
Expo Go
Figma

Breve descrição: Projeto mobile desenvolvido com React Native, utilizando o firebase como backend para autenticação e armazenamento de dados. Também foi utilizado o node.js para instalação do gerenciador de pacotes, onde foi utilizado o NPM para ajustar as dependências do projeto.
Expo Go foi utilizado para facilitar o desenvolvimento e testes do aplicativo em dispositivos móveis. Os primeiros design do aplicativo foi criado no Figma, garantindo uma interface amigável e intuitiva para os usuários.
